Software and Scientific Computing

At times solving problems in Physics, requires a bit of skill to get computers to do things for you. Though this section seems obvious to anyone doing physics, I included it to share things I have had chance to dabble with, in my research.   

  • Simulations/Numerics : MATLAB  |  Mathematica(especially for Symbolic calculations) 
  • Wolfram Alpha-Computational intelligence (Online tool for some quick math intuition)
  • Programming Languages:  Fortran | Python (Anaconda )  |  Julia 
  • Scientific Computing tools for Python(SciPy)
  • netlib- Fortran and C package library for science (Why we still use Fortran? )
  • LaTex  (for typesetting equations and drafting text).
    LaTex Editors: Texmaker, TeXstudio and Overleaf(online)
  • Reference-LAPACK on github
  • CTAN- Archive Network(a place to find LaTEX packages)
  • Text editors I've use(d): Sublime text | Microsoft VS code editor | Vim (Only for the Gods!)
